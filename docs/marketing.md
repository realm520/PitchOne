下面给你一份**去中心化链上足球博彩项目的冷启动作战手册**（不谈合规，只谈策略与机制），目标是在极小预算下快速形成**可下注的盘口+真实成交+可验证结果**的“最小可行生态”（MVE），并在 90 天内把**留存与口碑**做稳。

# 0. 冷启动的核心难题（用机制拆解）

1. **没人下注** ⇢ 先做“供给侧”：有深度的盘口与稳定赔率（LP/AMM 先行）。
2. **没人做预言机** ⇢ 激励上报与挑战、SLA 保障、错误罚没明确。
3. **没人来玩** ⇢ 有记分与奖励的赛事季 + 免佣券 + 可验证排行榜。
4. **没人信任** ⇢ 规则、赔率、结算、发奖都上链可查；每日公开仪表盘。

---

# 1. 产品切片：只做“一个联赛 + 单一玩法”的 MVE

* **联赛**：优先选流量稳定、时间密集的联赛（如英超/欧冠小组阶段其一）
* **玩法**：只做“90 分钟胜/平/负”（3 路），不含加时/点球；边界情形写死在 RuleSet V1。
* **网络**：L2（OP/Arb/zk 其一），目标**每笔下注 < $0.05 gas**。
* **定价**：采用 **三路 CPMM**（实现简单、gas 便宜），临近开赛提升费率与滑点保护。
* **代币**：Outcome 用 ERC-1155，LP 用 ERC-4626。

> 先把“**一条联赛主线×一个玩法**”跑通，等数据闭环稳定再扩品类。

# 2. 供给侧冷启动：先把“钱”和“结果”两头补齐

## 2.1 流动性（LP / 做市）

* **启动金库**：团队金库先注入 **$50k–$150k 稳定币** 作为 LP 种子（按热度分配到 3 路）。
* **LP 早鸟激励**：前 30 天 **交易费 100% 让利给 LP + 额外 10% LP Mining**（由 Promo Pool 出）。
* **深度目标**：热门赛前 24h 的**盘口深度 ≥ $20k**，单笔 $200–$500 的下注滑点 < 1.5%。
* **风控**：

  * 单地址持仓上限 ≤ 单场奖池 5%
  * 锁盘前 10 分钟禁“买入”，只许卖出或撤 LP

## 2.2 赛果（Oracle / Proposer）

* **Proposer 质押**：每场押金 =（奖池 × 1%）且 ≥ $300；正确上报返还并拿 **30% 押金收益**。
* **挑战者奖励**：成功挑战拿 **70% 错报罚金**。
* **SLA**：T_end + 30–60 分钟必须上报，否则开放“任意人上报”+ 优先奖励。
* **储备人力（仅应急）**：内置 2–3 个“受托上报人”钱包，只在无人上报时触发；过程仍走押金与挑战流程，**不形成中心化裁判**。

# 3. 需求侧冷启动：先把“玩的人”聚起来

## 3.1 三阶激励漏斗（30 天滚动）

1. **到店礼**：新钱包首笔下注返 **20% 手续费抵扣券**（Credit Token，30 天有效，单笔最多抵 50% 手续费）。
2. **留存礼**：连续 7 天参与任意 1 场，送“串关+0.5% 赔率加成券”1 张。
3. **赛季榜**：按**有效投注额（对数缩放）×参与天数**计分，Top 10% 领取赛季 NFT + 返现空投。

## 3.2 引荐（和你上一轮的推广机制对齐）

* **一次绑定，90 天分成**：推荐人返佣 8% 起，被推荐人返利 2% 起；当周预算不够按比例缩放。
* **升级门槛**：30 天新增 20 个有效用户 ⇒ 10% 返佣；100 个 ⇒ 12% + SBT 权限。
* **低成本素材**：邀请码链接 + 批次化 Referral NFT（送免佣券）。

## 3.3 社区合作

* **目标 KOL**：加密足球迷/KOL、Fantasy 足球群、DAO 社区体育频道。
* **玩法**：“KOL 主题页”列出他当周心水比赛，**从主题页成交的返佣+1%**（Promo Pool 出）。
* **线下观赛**：酒吧/球吧 QR → 绑定推荐 + 领免佣券；每家店设置“战队榜”，赛季发宝箱。

# 5. 0→1 的时间表（T 表示正式上线日）

## 预热期（T-30 ~ T-7）

* 选定 10–20 场“标杆比赛”（一条联赛），在 **Testnet 做仿真赛季**：

  * 白名单用户参与，送测试券，验证 AMM 曲线/风控参数。
* 招募 **首批 Proposer 10–20 人**（质押金由项目临时补贴 30%），跑通上报/挑战流程。
* 与 10 个以上 KOL/社群确定“主题页+抽奖”活动 slot。

## 上线首周（T 到 T+7）

* 仅上 **3–5 场** 热门比赛，**盘口深度集中**，保证体验。
* 开启 **“七日打卡”活动** + 新人 20% 免佣券。
* 每日在 X/Telegram 公示前一日数据卡：成交、上报时延、Merkl e 发放哈希。

## 扩品期（T+8 ~ T+30）

* 扩到每周 15–25 场（同一联赛），上线 **串关 2×**。
* 开启 **LP Mining** 与 **排行榜赛季制**，首季 30 天。
* 引入 **战队模式**：每个推荐人可建战队，赛季按队分结算宝箱。

## 放大期（T+31 ~ T+90）

* 扩展到第二条联赛或第二种玩法（让球/大小球二选一）。
* 推出 **Promoter SBT** 与 **活动工厂**（UGC 主题页自助）。
* 上线 **反作弊评分**（链上图谱 + 护照分）与 **线性释放**（返佣 T+7 解锁）。

# 6. 冷启动预算与指标（建议值）

* **种子 LP**：$50k–$150k
* **Promo Pool（首季）**：$20k–$50k（来自交易费分成 + 预留）
* **KOL/活动物料**：$5k–$10k 等值券（尽量用免佣券、加成券而非现金）

**核心 KPI（周维度）**

* WAU ≥ 500、GTV ≥ $150k、留存 D7 ≥ 25%
* 预言机上报中位时延 ≤ 30 min，挑战率 2%–8%，错误上报纠正率 ~100%
* LP APR（年化）≥ 15%（费率 + Mining）
* CAC/LTV ≤ 0.6（按返佣与返利摊销测算）

# 7. 现场运营与风控手册（冷启动版）

* **限额**：新用户首周单场投注上限 $200；热门赛上限逐步抬升。
* **动态费率**：开赛前 60→10 分钟费率从 0.75% 线性升至 1.5%。
* **异常报警**：

  * 单地址/小团体对手盘重合度 > 70% → 降权，不计入返佣
  * 单场在 5 分钟内注入 > 奖池 20% → 自动拉高滑点与费率
* **应急**：无人上报超时 → 触发开放上报 + 双倍奖励；连续争议失败的 Proposer 自动拉入冷却名单。
